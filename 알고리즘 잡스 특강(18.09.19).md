#### tobin

- 재귀함수 문제.

- 입력받는 것에 따라 for문의 중첩 수가 결정되는 문제는 재귀를 사용해야함.

  - 재귀함수 풀때
    1. 매개변수
    2. 탈출구
    3. 구현
  - 매개변수 (길이, 1개수)
  - 배열에 값을 가지고 있어야 출력할 수 있으니 저장할 필요가 있음.

  ```
  tobin(l, o) {
      if (l == n) {
          if (o == k) {
              arr[l] = 1;
              tobin(l+1, k+1);
              arr[l] = 0;
              tobin(l+1, k);
          }
      }
  }
  ```
